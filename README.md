# Tom-Tom — A Jornada do Thor

> Jogo web de plataforma procedural em HTML5 Canvas, tema Thor / Tom-Tom.

## Sobre

Mesma dinâmica de plataforma side-scrolling: colete moedas de ouro, quebre blocos, desvie dos inimigos e atravesse o portal. Fases procedurais com clima, estação, trilha e save local.

## Como jogar local

Abra `redobrinha-game/index.html` no navegador, ou sirva a pasta:

```bash
cd redobrinha-game
npx serve .
```

## Deploy (Netlify)

Publish directory: `redobrinha-game` (definido em `netlify.toml`).

## Controles

- ← → / A D — mover  
- ↓ / S — abaixar  
- Espaço — pular (duplo)  
- Shift — correr  
- Touch nas laterais no celular
